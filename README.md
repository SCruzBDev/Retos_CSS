## De que trata

Flexbox Froggy es un juego interactivo en el que se ayuda a un grupo de ranas
a llegar a sus nenufares. Su proposito es ensenar el modelo Flexbox de CSS, un
sistema de maquetacion pensado para distribuir y alinear elementos a lo largo
de una sola dimension (una fila o una columna), adaptandose de forma flexible
al espacio disponible.

## Objetivo a cumplir

El reto consiste en escribir las propiedades de Flexbox necesarias para mover
cada rana hasta el nenufar de su mismo color. A medida que avanzan los niveles
aparecen varias ranas y nenufares en distintas posiciones, lo que obliga a
combinar varias propiedades para lograr la distribucion exacta.

## Como se juega

1. Un contenedor (el estanque) alberga una o varias ranas.
2. El nivel muestra donde deben terminar las ranas.
3. Se aplican propiedades de Flexbox al contenedor (y a veces a las ranas).
4. Cuando todas las ranas quedan sobre su nenufar, se supera el nivel.
