## De que trata

CSS Diner es un juego interactivo, ambientado en una mesa de restaurante con
platos, manzanas y cubiertos, cuyo proposito es ensenar a escribir selectores
de CSS. Los selectores son la base de todo el lenguaje: sin ellos no es posible
aplicar estilos, porque son el mecanismo con el que se le indica al navegador
a que elementos debe afectar una regla.

## Objetivo a cumplir

El reto consiste en escribir, en cada uno de los niveles, el selector exacto
que apunte unicamente a los elementos indicados sobre la mesa. Si el selector
es demasiado general seleccionara elementos de mas, y si es demasiado
especifico dejara elementos fuera. El desafio esta en encontrar la expresion
mas precisa posible para cada situacion.

## Como se juega

1. Se muestra una mesa con distintos platos y alimentos representados como
   elementos HTML.
2. El nivel indica cual o cuales elementos hay que seleccionar.
3. Se escribe el selector en el editor y el juego resalta lo seleccionado.
4. Si la seleccion coincide con lo pedido, se avanza al siguiente nivel.
